# 🎲 Dice Game

A simple and fun Dice Game built using **HTML, CSS, and JavaScript**.
Refresh the page to roll the dice and see who wins!

---

## 🚀 Features

* 🎲 Random dice roll for two players
* 🖼️ Dynamic dice images update
* 🏆 Winner announcement logic
* 🤝 Draw condition handling
* ⚡ Lightweight and beginner-friendly project

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (Vanilla JS)

---

---

## 🎮 How to Play

1. Open the project in your browser
2. Refresh the page
3. Both dice will roll automatically 🎲
4. The player with the higher number wins 🏆

---

## 🧠 Concepts Used

* `Math.random()` for generating random numbers
* DOM Manipulation using `document.querySelector()`
* Updating attributes using `setAttribute()`
* Conditional statements (`if-else`) for game logic
